# Omron Import

Importiert automatisch Blutdruckmessungen aus OMRON Connect
in eine Excel-Auswertung.

## Features

- automatische CSV-Erkennung
- Excel COM
- keine beschädigten Formeln
- Duplikaterkennung
- Backup
- Archivierung

## Windows

Python 3.14

pip install -r requirements.txt

start_import.bat
